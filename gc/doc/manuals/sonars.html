sonar

The SONAR server process that talks to the I2C-to-RS232 converter that is connected through a distribution board to Devantech SRF08 sonars.

Syntax:

sonar [option[,option ...]] ... &

Runs on:

X86, QNX Neutrino, needs watchdog timer to be running for communication to happen

Options:

-h | --help Displays help

-V | --version Display version

-v | --verbose Sets verbose flag

-d | --dev serial_dev Serial device to use to talk to the I2C-to-RS232 converter

Description:

The sonar server talks to each of the SRF08 units through a I2C-to-RS232 converter and a I2C distribution board. It repeatedly asks each SRF08 to send out a ping and then asks it for the distance estimation. It cycle through 6 of the SRF08 in about 1 second and so the update rate of the distance estimation is about 1hz. Refer to Note 43 on the Overbot website for more elaboration.

Usage:

sonar -h

or sonar --help

will both display help

sonar -d /dev/ser2

Will use /dev/ser2 to talk to the I2C-to-RS232 converter.

Messages:

Refer to /gc/src/qnx/common/include/sonar_messaging.h for the communication interface. However, the sonar server has not been implemented as a server yet. So the sonar_messaging.h file is still tentative.
