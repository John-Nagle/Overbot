gpsins_gui

A GUI that talks to the GPSINS server process and draws the data on the screen using photon.

Syntax:

gpsins_gui &

Runs on:

X86, QNX Neutrino, needs watchdog timer to be running for communication to happen

Options:

NONE

Description:

The gpsins_gui is a pure client that only talks to the GPSINS server process. It polls for data regularly and updates its drawing. It is implemented in Photon, C++. It features an artificial horizon display of attitude, a top view position display and a simple compass heading display.

Usage:

gpsins_gui
